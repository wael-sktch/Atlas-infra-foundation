# DC01 - Domain Controller

## Overview

DC01 is the Primary Domain Controller for the Atlas Infrastructure Foundation lab.

## Host Information

Hostname: DC01

Operating System:
Windows Server 2022

IP Address:
10.10.20.10

Subnet Mask:
255.255.255.0

Gateway:
10.10.20.1

DNS:
10.10.20.10

## Roles Installed

- Active Directory Domain Services
- DNS Server

## Domain

atlas.local

## Purpose

Provide:

- Authentication
- Domain Services
- DNS Services
- User Management
- Group Policy
